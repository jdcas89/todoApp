## React Native Todo App

## Installing dependencies

- Node js ( 8.5.0 +)
- Xcode ( for iOS )
- Android Studio ( for Android )

## NPM packages 
-   React Native CLI

```
npm install -g react-native-cli
```

## App installation

```
npm install
```

## Running the app locally


```
npm run start
```


## Running the app on your device

```
npm run serve-ios
npm run serve-android

```
